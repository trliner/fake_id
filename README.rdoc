= FakeId

http://github.com/trliner/fake_id

== Usage

=== Defining fake ids

  class SecretAgentAlias
    include FakeId

    attr_accessor :document_type_id

    fake_ids_for :document_type, {
      :drivers_license => 1,
      :passport => 2
    }
  end


=== Generated instance methods

  alias = Alias.new(:document_type_id = 1)
  alias.document_type
  => :drivers_license

  alias.document_type = :passport
  alias.document_type
  => :passport
